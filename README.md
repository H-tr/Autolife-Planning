# Autolife-Planning

This is a library for planning in with Autolife robot.

## Installation

```bash
git clone --recursive https://github.com/H-tr/Autolife-Planning.git
pip install -e .
```

Download the assets

```bash
bash scripts/download_assets.sh
```

## Usage

Run some example scripts first:

```bash
python examples/random_dance_around_table.py
```

## Robomesh

If you want to run it in robomesh, please check the [readme in the doc folder](docs/README.md).
